# VacunApp

[Versión en español](README.md)

Mobile app built with Flutter to view, display, and manage digital vaccination cards.

The project was created as an open proposal to explore a simple digital card experience: vaccination cards per person, vaccination certificates, and QR code scanning.

## Features

- View vaccination cards by person.
- Carousel navigation between registered cards.
- Screen to add a new vaccination card.
- Vaccination certificate view.
- QR code scanner.

Run the app:
